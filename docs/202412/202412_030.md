2. ：平台，加上美国人热衷于个性化产品，所以平台产品有。 3. ：平台收取佣金=交易金额的 6.5%，其他平台在 30%左右。 4. ：因为平台主打手工定制，同质化没那么严重，竞争压力相对较小，只要做好产品图片和内容就可以。
