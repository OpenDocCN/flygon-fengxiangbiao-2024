3、会告诉用户，自己准备了一个本子，把前 7000 位关注的“原始股东”小红书 ID 都记录下来，未来进行回馈。形成明确的行动指令，引导关注；
4、持续发布视频，从关注者中抽奖，奖品即为商单。
