正文：

ChatGPT 偷懒？假装给点小费就好 最近 Christian Mayer 博士利用 OpenAI API 和 GPT-4 Turbo
模型对这个传闻进行了实验。他设计了一系列提示，每个提示中包含不同的小费激励，范围从零到一百万美元不等。
测试结果表明，小额打赏可能会带来明显的积极影响，但打赏的金额似乎也很关键。 1️⃣ 0.1 美元效果最差 2️⃣ 10 美元性价比最高 3️⃣ 10
万-100 万美元之间，钱越多效果越好
