# 快手游戏合伙人计划又有搞头了

正文：

快手游戏合伙人计划又有搞头了，我看了一下快手游戏合伙人周榜还是有很多人一个视频就能有过万的收益，虽然比不上当时的抖音游戏发行人计划，但现在一个视频能赚这么多钱还是挺不错的，还是有很多手游在快手推广，这些收益高的基本上都是按照 CPA 下载结算收益，还有有很大的机会
项目玩法是在快手搜索“游戏合伙人”，然后在这里面做任务赚取收益，最好做这蛋仔派对、圆梦之星、萤火突击这些游戏，因为这些游戏单价高，我分别还搜索了这周三个游戏的推广第一名，这三个账号都是新号，一个视频就赚大几千好几万，由此可见这快手合伙人计划还有搞头
