# 转发工作机会与优惠信息，小红书账号吸引粉丝并实现变现

正文：

转发各种工作机会，发在小红书上，也能收获很多粘性很好的粉丝。
像是这个账号，转发一些国企招聘，展会兼职，街道招聘，还有政府发的优惠券信息，可见投稿 78 条，获邀 1.4 万，粉丝 1.5 万。
后期变现，卖会员，给其他小程序做导流，帮企业招聘等。
