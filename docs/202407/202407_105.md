# 快手游戏合伙人萤火突击 CPA 提升至 9-12 元，收益潜力增加

正文：

快手游戏合伙人这个萤火突击这个游戏 CPA 提升到 9-12 块钱，最近这两天很多人在做这个游戏，当时这个游戏 CPA 好像是 6 块钱左右，那时候收益榜单最高的一条视频是 4.6 万收益，现在 CPA 上调了收益会更加高，而且还上不封顶，想做这个的一定要认真看一下活动规则，避免出现问题导致收益被扣除，感兴趣的可以玩一玩
