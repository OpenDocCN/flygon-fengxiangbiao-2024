# 爆改盲盒：小众行业中的高利润服务

正文：

小红书、咸鱼平台上，由盲盒衍生出来的小众行业——爆改盲盒。
盲盒玩家在抽到盲盒之后，送到“改娃师”手上进行二创，或统一改成复杂款，或单独定制个人款，价格也根据款式的繁简程度而不同。
盲盒本身价值在百元以内，使用到的改造材质多为粘土，成本不高，而爆改手工费高达几百到几千不等。属于小众、低成本但高利润的服务，受众群体多为 90、00 后。
